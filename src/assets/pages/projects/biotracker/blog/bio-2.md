# (Painting & Electronics)

---

Once all the components had been assembled and glued together, sanded and polished, the next step would be to paint it. Not all paints stick to PETG plastic, so some experimentaion would need to be done in order to find the best option. To begin with I tried using some old Citadel warhammer paints, however these usually require a primer and even with that they did not properly adhere to the plastic. The next option was using some old car spraypaint, this adhered well, and additionally would function as a primer for the warhammer paint I intended to use for decoration. Using maskng tape to protect areas that would not need painting, the whole project was coated in a dark grey cover, then to go for the used and roughed up metal look the game's model has, I used a silvery Citadel paint along the edges.

Finally was the electronics, initially I wasn't going to incorperate electronics onto the project, so a few pieces had to be reworked and 'modified' with a heated knife to make space for wiring and switches. The result includes a main switch, a trigger, 4 LED panels in the head, and a detachable battery pack all wired together to produce a torch effect at the front activated by pulling the trigger. At the top I intended to add a laser and another light however these did not make it into the final design. Additionally, there is a removable old phone built into the screen portion of the project, which I intend to have display an animated visual of what appears in the game.

---

# (Summary)

Overall the project turned out rather well, however the main difference I would have made early on was choosing not to scale the whole thing up by 25%, as it turned out much larger than anticipated. The ingame model is deceptively small, so while scaling up the project makes it fit better with real world items like the grip, bolts and switches, it is definitely a bit too large. The aesthetics turned out very well I feel, and being able to reproduce something from an ingame model into real life was a very enjoyable experience. Even though there were issues I have learnt a lot and will continue to create things using these lessons.
